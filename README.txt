Provides widgets to display the status of a subtitling project.

== Description ==  
Currently provides two widgets:

1. Single episode widget - displays the status of any single episode, regardless of the visibility flag
1. All episodes widget - lists all episodes that have the visible flag enabled in the admin.

See the Github page for roadmap and bug reports.
http://github.com/precurejunkie/subtitle-status/wiki

== Installation ==

1. Place the unpacked "subtitle-status" directory in the "/wp-content/plugins/" directory.
1. Activate the plugin through the "Plugins" menu in WordPress.
1. Go to "Subtitle Status" under "Settings" and create at least one series and add an episode to it
1. Edit the episode to set the status of each step appropriately
1. Go to the "Widgets" page and put the widget where you want it.
